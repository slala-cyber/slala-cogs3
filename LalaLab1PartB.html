What is DOCTYPE statement?
- It is not an HTML tag, but rather provides information to the browser about the specific type of document to expect
How do you use:
- dl Defines a description list
- ul Defines an unordered list
- ol Defines an ordered list
Difference between text editor and word processor?
- Word processors work with formatted text while text editors work with plain text
What is the client-server model?
- A network architecture where a client sends a request to a server to access their requested resource or service
Provide a description and definition for the following network protocols
- FTP: File Transfer Protocol, used for the transfer of computer files from a server to client on computer network
- HTTP: Hypertext Transfer Protocol, used to transfer web pages between web browsers and servers
- TCP/IP: Transmission Control Protocol or Internet Protocol, a fundamental protocol for defining how data is transmitted, routed, and addressed across networks in small managable packets
